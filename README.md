# Project Approval & Management System

A role-based web application developed using **Python Flask** and **MySQL** to streamline academic project submission, review, approval, and management processes within educational institutions.

## 📌 Overview

The Project Approval & Management System provides a centralized platform where students can submit project proposals, teachers can review and manage projects, and administrators can oversee the entire approval workflow.

The system reduces manual paperwork, improves communication between students and faculty, and enables efficient tracking of project status throughout the project lifecycle.

---

## 🚀 Features

### 👨‍🎓 Student Module

* Student registration and login
* Submit project proposals
* Upload project-related documents
* Track approval status
* View project details
* Communicate with project supervisors

### 👨‍🏫 Teacher Module

* Teacher authentication
* Review submitted projects
* Approve or reject project proposals
* Manage assigned students
* View project submissions
* Provide feedback and comments

### 👨‍💼 Admin Module

* Manage students and teachers
* Monitor project activities
* View all project submissions
* Manage user accounts
* Track project approval workflows

## 🏗️ System Architecture

1. Users authenticate through role-based login.
2. Students submit project proposals and documents.
3. Teachers review submitted projects.
4. Projects are approved, rejected, or sent back for modification.
5. Administrators monitor and manage the overall workflow.
6. All project information is securely stored in the MySQL database.
